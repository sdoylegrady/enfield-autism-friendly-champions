# Enfield Autism-Friendly Champions

A starter website for an Enfield Autism-Friendly Champions community map.

## How to edit the champions

Open `champions.js` and replace the sample entries.

Each champion needs:

- name
- category
- status
- address
- supports
- description
- latitude
- longitude
- website

## How to publish on Vercel

1. Create a GitHub account or sign in.
2. Create a new repository called `enfield-autism-friendly-champions`.
3. Upload these files.
4. Go to Vercel.
5. Import the GitHub repository.
6. Deploy.

## Notes

This project uses OpenStreetMap and Leaflet.
It does not use puzzle-piece imagery.
